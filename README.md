# Obsidiana Multimídia - Gestão de Inventário e Orçamentos
📌 Sobre o Projeto

Este projeto foi desenvolvido para atender às necessidades da Obsidiana Multimídia, uma empresa que realiza cobertura de eventos e serviços de multimídia.
Por estar em fase inicial, a empresa identificou a necessidade de uma ferramenta que:

Facilite o controle do inventário de equipamentos;

Gere orçamentos personalizados de acordo com os serviços e equipamentos necessários;

Garanta maior agilidade e organização nos processos internos.

🚀 Tecnologias Utilizadas

O sistema está sendo construído utilizando uma stack moderna, visando escalabilidade e manutenção a longo prazo:

Backend: Java com Spring Boot

Frontend Web: React.js + Node.js

Banco de Dados: SQL relacional

Mobile (futuro): Versão híbrida/nativa (a definir, ex.: React Native ou Flutter)

🗂️ Estrutura do Projeto

/backend        -> Código Java com Spring Boot  
/frontend       -> Aplicação web em React.js  
/database       -> Scripts e modelagem SQL  
/mobile         -> (planejado) versão mobile do sistema  

🎯 Funcionalidades

 Cadastro e gerenciamento de equipamentos

 Controle de estoque/inventário

 Criação de orçamentos personalizados

 Autenticação de usuários

 Dashboard com indicadores de uso e status dos equipamentos

 Integração com a versão mobile

📊 Modelagem de Dados

A modelagem segue padrão relacional (SQL), garantindo integridade e consistência nas informações de inventário e orçamentos.

🔮 Próximos Passos

Disponibilizar a versão mobile do sistema

👥 Equipe

Projeto desenvolvido por estudantes de Análise e Desenvolvimento de Sistemas, com foco em aplicar boas práticas de engenharia de software e atender às necessidades reais da Obsidiana Multimídia.
